# SEO Copywriting Framework

## Keyword Research Process
```
1. Seed Keywords → Brand-relevant terms
2. Expand → Use tools (Ahrefs, SEMrush, etc.)
3. Filter → Search volume, difficulty, intent
4. Prioritize → Business value + brand alignment
5. Map → Assign to content types/pages
```

## Content Structure Template
```markdown
# [H1: Primary Keyword + Compelling Hook]

[Opening paragraph: Hook + Primary keyword + Value proposition]
~100 words establishing context and reader benefit

## [H2: Secondary Keyword / First Main Point]
[Content addressing search intent]
[Include related keywords naturally]

### [H3: Supporting detail]
[Expand on subtopic]

## [H2: Second Main Point]
...

## [Conclusion / CTA Section]
[Summarize value]
[Clear call to action]
[Internal links to related content]
```

## Meta Tag Guidelines

### Title Tag
```
Format: [Primary Keyword] - [Benefit/Modifier] | Studio Sen
Length: 50-60 characters
Example: Brand OS Platform - Unified Brand Management | Studio Sen
```

### Meta Description
```
Format: [Value prop] + [Primary keyword] + [CTA]
Length: 150-160 characters
Example: Manage your brand intelligence in one place. Studio Sen's Brand OS
         ensures consistency across all touchpoints. Explore the platform →
```

### Header Hierarchy
```
H1: One per page, primary keyword, compelling
H2: Section headers, secondary keywords
H3-H6: Supporting structure, LSI keywords
```

# Brand Voice in SEO Copy

## Studio Sen Voice Attributes
```yaml
tone: Professional yet approachable
style: Clear, concise, authoritative
vocabulary:
  use: Industry terms appropriately, active voice
  avoid: Jargon overload, passive constructions, hyperbole
sentence_structure:
  - Vary length for rhythm
  - Lead with benefit statements
  - End paragraphs with forward momentum
```

## Voice Examples

### On-Brand SEO Copy
```
"Brand consistency shouldn't require constant vigilance. Studio Sen's
Brand OS automates compliance checking, so your team can focus on
creating—not policing."
```

### Off-Brand (Avoid)
```
"AMAZING brand management solution!!! Stop worrying about brand
consistency forever with our revolutionary platform that will
transform your business!"
```

## Keyword Integration Rules
- Primary keyword: H1, first paragraph, naturally throughout
- Secondary keywords: H2s, body content
- LSI keywords: Support natural language variety
- **Never**: Keyword stuff, force unnatural phrasing, sacrifice readability

# SEO Checklist

## On-Page SEO
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Keyword density 1-2% (natural)
- [ ] Secondary keywords in H2s
- [ ] Meta title optimized (50-60 chars)
- [ ] Meta description compelling (150-160 chars)
- [ ] Image alt text includes keywords
- [ ] Internal links (3-5 per page)
- [ ] External links to authoritative sources

## Content Quality
- [ ] Answers search intent completely
- [ ] Provides unique value/angle
- [ ] Accurate, up-to-date information
- [ ] Proper grammar and spelling
- [ ] Scannable (headers, bullets, short paragraphs)
- [ ] Mobile-friendly formatting

## Brand Alignment
- [ ] Voice matches brand guidelines
- [ ] Tone appropriate for content type
- [ ] No prohibited words/phrases
- [ ] CTAs align with brand messaging
- [ ] Visual content follows brand style
